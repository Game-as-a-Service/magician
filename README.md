# Vue 3 + Vite 模版

## 建議安裝插件
1. [blueprint](https://marketplace.visualstudio.com/items?itemName=teamchilla.blueprint)
    - 可以快速產生vue component或vue page架構
2. [code spell checker](https://marketplace.visualstudio.com/items?itemName=streetsidesoftware.code-spell-checker)
    - 檢查拼字是否正確
3. [eslint](https://marketplace.visualstudio.com/items?itemName=dbaeumer.vscode-eslint)
    - 對於js程式碼的檢查閱讀性
4. [Path Intellisense](https://marketplace.visualstudio.com/items?itemName=christian-kohler.path-intellisense)    
    - 在匯入各檔案時會有根目錄提示出現
5. [stylelint](https://marketplace.visualstudio.com/items?itemName=stylelint.vscode-stylelint)
    - 對於css語法的檢查及閱讀性
6. [auto rename tag](https://marketplace.visualstudio.com/items?itemName=formulahendry.auto-rename-tag)
    - 對html tag在改寫時能夠自動抓取對應的tag
7. [Vue Language Features (Volar)](https://marketplace.visualstudio.com/items?itemName=Vue.volar)
    - 在編輯器上針對vue3的提示

## 內含套件
1. axios
2. vue-router
3. pinia
4. husky
5. normalize.css
6. eslint
7. stylelint
